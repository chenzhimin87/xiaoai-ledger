# 🌸 温暖手账 - 智能日程管理

一款温暖柔和风格的智能电子手账，支持文字识别、图片上传、工作/生活双区管理。

## ✨ 功能特点

### 🎯 智能识别
- **文字识别**：粘贴会议通知、材料提交要求等文字，自动解析标题、时间和分类
- **图片上传**：支持拖拽或点击上传截图（会议PPT、聊天记录等）
- **时间解析**：智能识别"明天下午3点"、"下周二"、"周五前"等自然语言时间

### 📋 双区管理
- **💼 工作区**：会议、报告、项目、提交材料等工作相关日程
- **🏠 生活区**：约会、生理期、健身、针灸推拿等生活事项

### 📊 统计面板
- 实时显示待办工作数量
- 实时显示生活事项数量
- 今日待办提醒

### 🎨 温暖柔和风格
- 奶油白主背景，温暖桃色强调
- 工作区柔和蓝色，生活区柔和珊瑚粉
- 圆角设计 + 柔和阴影

### 📱 多端适配
- 完美适配 Windows 电脑
- 完美适配鸿蒙手机（华为）
- 响应式布局，手机电脑都能用

## 🚀 使用方法

### 在电脑上使用

1. **打开网页**：在浏览器中访问手账链接
2. **智能识别**：
   - 复制会议通知或材料要求
   - 粘贴到"智能识别"文本框
   - 点击"识别日程"按钮
3. **查看结果**：系统自动填充标题、分类、截止时间
4. **保存日程**：确认无误后点击"保存日程"

### 在手机上使用

1. **添加到主屏幕**（鸿蒙/安卓）：
   - 用浏览器打开手账链接
   - 点击菜单 → "添加到主屏幕"
   - 像使用APP一样打开

2. **同步数据**：
   - 电脑和手机使用同一个链接访问
   - 数据存储在本地，换设备需手动同步

### 支持的识别格式

**时间表达**：
- 明天下午3点
- 后天上午10点
- 下周二晚上7点
- 周五前提交
- 4月15日下午
- 2026-04-20 14:30

**自动分类**：
- 工作关键词：会议、报告、项目、提交、截止、部门、客户等
- 生活关键词：约会、聚餐、健身、生理期、针灸、推拿等

## 💡 使用技巧

1. **快速添加**：点击右下角的"+"按钮快速新建日程
2. **完成任务**：点击任务左侧的复选框标记完成
3. **编辑删除**：鼠标悬停任务显示编辑和删除按钮
4. **紧急提醒**：24小时内的任务会显示红色时间标签

## 🔒 隐私说明

- 所有数据存储在浏览器本地（LocalStorage）
- 不上传任何数据到服务器
- 清除浏览器数据会导致手账内容丢失
- 建议定期导出重要日程

## 🛠️ 技术信息

- **类型**：PWA（渐进式Web应用）
- **技术**：HTML5 + CSS3 + JavaScript
- **存储**：LocalStorage 本地存储
- **适配**：响应式设计，支持各种屏幕尺寸
