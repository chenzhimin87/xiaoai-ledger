# 温暖手账项目记忆

## 项目概述
- 项目名称：小爱酱账（原名温暖手账）
- 目标：智能电子手账，自动识别日程并按工作/生活区块展示
- 工作目录：`d:\workbuddy工作区\电子手账\`

## 技术栈
- 纯前端实现（HTML + CSS + JS）
- 本地存储（localStorage）
- PWA支持（Service Worker）
